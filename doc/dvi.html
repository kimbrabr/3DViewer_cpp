3DViewer v2.0

О программе

3DViewer позволяет просматривать и взаимодействовать с трехмерными моделями.

Возможности программы

  • Работа с файлами *.obj
  • Перемещение модели по осям X, Y, Z
  • Поворот модели относительно осей X, Y, Z
  • Масштабирование модели

Установка и запуск

Для установки используйте команду: make install

Удаление программы

Для удаления используйте команду: make uninstall
